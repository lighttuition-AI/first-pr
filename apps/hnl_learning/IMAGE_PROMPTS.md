# HNL Learning — Picture prompts (replace the emoji placeholders)

Generate real pictures for these, then we slice each grid and bundle them (same as the story pictures).

## How to use
1. Paste the **Style block** once into ChatGPT.
2. Paste **one section's list** → ChatGPT returns a numbered grid in the SAME order.
3. Send me the grid; I slice it in order and replace those emojis in the app.
- Big sets (animals): do it **in batches of ~12** per grid so each picture stays sharp.

## 🎨 Style block (paste first)
> Make a single image: a grid of cute, friendly, **semi-realistic cartoon** pictures for a children's app (ages 2–8). Each cell = ONE subject, centered, full-color, on a **plain white background**, clearly recognizable, no text/labels/numbers. Keep them in the exact numbered order I give.
