# design-refs/

Drop your design pictures here so Claude Code can read and build from them:
screenshots, phone photos, Claude Design exports, sketches — anything visual.

**How to use it:**
1. Save your image in this folder (e.g. `home-screen.png`).
2. Tell Claude the filename: *"use the layout in design-refs/home-screen.png"*.

Reference images live here (alongside the code, version-controlled in Git) so we
always remember *why* a screen looks the way it does.

See `IDEA-INTAKE-CHECKLIST.md` for what to send with each new idea.
